CUSTOMISATION
--------------------------

1. Color Schemes
--------------------------
The theme offers 5 different colour schemes (@see: skins.htm file):
Blue: #377CA8
Red: #980000
Green: #81994d
Grey: #515257
Purple: #7F0046

Each one has a separate .css file within the /css directory:
1. colour-blue.css (default no need to include file)
2. colour-red.css
3. colour-grey.css
4. colour-purple
5. colour-green.css

These files can be used as an example of how to implement your own colour schemes.

Example - Add the following to your <head> tags below the "Required CSS": <link href="css/colour-green.css" rel="stylesheet">

NOTE: You will need to do this on all pages to implement site wide.

2. Overriding Other Styles
--------------------------
The theme also offers & includes automatically a .css skeleton file called custom-style.css which can be used to override the theme structure & media queries.
This file is found within the /css directory and is well commented to provide instruction.


3. Adding New Pages
--------------------------

The theme comes with a starter template (starter.htm) which can be copy & pasted to start new pages.
